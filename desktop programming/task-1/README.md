\f0\fs24 \cf0 # Task 1 - Restaurant Management (Basic OOP)\
\
This task is part of **Desktop Programming**.\
\
## \uc0\u55356 \u57263  Learning Objectives\
- Implement **classes and objects** in Java\
- Use **arrays** to store data\
- Apply **conditional structures** (`if`, `else`, `switch`)\
- Write **methods** for calculating subtotal, discount, and tax\
\
## \uc0\u55358 \u56809  Task Description\
Create a **restaurant order management program** that:\
1. Displays food and drink menus.\
2. Allows the user to input up to 4 orders.\
3. Calculates total cost, 10% tax, and service fee.\
4. Adds promotions:\
   - 10% discount if total > Rp100.000  \
   - \'93Buy 1 Get 1 Free\'94 for drinks if total > Rp50.000\
\
