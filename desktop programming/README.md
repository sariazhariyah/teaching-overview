\f0\fs24 \cf0 # \uc0\u55357 \u56507  Desktop Programming (STSI4201)\
\
This repository contains **tutorial materials and code skeletons** for the course **Pemrograman Berbasis Desktop**\
\
The goal of this repository is to provide **structured examples** and **template code** that help students learn Object-Oriented Programming (OOP) concepts and Java fundamentals through practical, console-based applications.\
\
---\
\
## \uc0\u55357 \u56538  Course Overview\
The course introduces the fundamentals of desktop-based programming using **Java** as the main language.  \
Students will explore:\
- Core Java syntax (data types, variables, conditions, and loops)\
- Object-Oriented Programming (OOP) principles\
- Arrays and string manipulation\
- Exception handling and file I/O\
- Application structure using classes and methods\
\
---\
\
\
## \uc0\u55356 \u57263  Learning Outcomes\
After completing all tasks, students should be able to:\
1. Implement **object-oriented design** using classes, objects, methods, and attributes.\
2. Apply **decision structures** (`if`, `switch`) and **looping** (`for`, `while`, `do-while`) effectively.\
3. Handle **arrays**, **exceptions**, and **file operations**.\
4. Develop small but complete **console-based applications** following clean code practices.\
\
---\
\
